# Web Frontend - Artisan Breads

Web frontend for the Artisan Breads system, designed for GitHub Pages deployment.

## Setup

1. Update API base URL in `app.js`:
   ```javascript
   window.BASE = 'http://your-backend-url:8000';
   ```

2. Deploy to GitHub Pages:
   - Push to GitHub repository
   - Enable GitHub Pages in repository settings
   - Select branch and folder

## Pages

- `index.html` - Home/Bread listing
- `login.html` - Login
- `register.html` - Registration (customer only)
- `profile.html` - User profile
- `admin.html` - Admin dashboard
- `artisan.html` - Artisan dashboard
- `customer.html` - Customer dashboard
- `cart.html` - Shopping cart
- `orders.html` - Order history
- `bread-detail.html` - Bread details
- `verify-email.html` - Email verification
- `forgot-password.html` - Password reset
- `reset-password.html` - Password reset form

## Technologies

- HTML5
- CSS3 (Tailwind CSS)
- JavaScript (Vanilla JS + jQuery)
- SweetAlert2
